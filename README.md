# ProjectX-FoodsDetection
料理の物体検出を行うソースコード

## 物体検出
料理最終.ipynbの環境リセットからバウンディングボックス通りにトリミングまでが物体検出のソースコード  
料理の画像から各料理を検出して各料理画像にトリミングを行う。  
実行内容が複雑な為詳しくは[こちら](link)を参照。  
フォルダの階層は個人によって違うため注意してください。
